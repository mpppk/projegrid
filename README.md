# projegrid

```
vim .env
vim config/config.js
```

## Start server
```
npm install
npm start
```

## Start with heroku
### Requirement
* heroku-toolbelt

### Command
```
heroku local
```

### Note
.envファイルの反映はheroku経由でしか行われないので注意

## Start dev server

```
npm install
npm run dev-server
```

http://localhost:3000
